# FNAFCOOP1OMNIMAN
Omni-Man script for Roblox FNAF Features: Auto Doors: Closes based on Bonnie/Chica's X and Z coordinates and Foxy's distance. ESP Scan: Highlights all enemies. Full Bright: Night light. Anti-Golden: Automatically runs gold.Tactics: Keep the camera on Foxy at all times and check Freddy's ESP to freeze them completely.
